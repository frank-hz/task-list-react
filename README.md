## Setup
```bash
# Clonar el repositorio
$ git clone https://github.com/frank-hz/task-list-react.git

# Ir a la carpeta
$ cd task-list-react

# Instalar dependencias
$ npm install

# Ejecutar aplicacion
$ npm run start
```
